# EcoPackAI-AI_Powered_Sustainable_Packaging_Recommendation_System
EcoPackAI is an AI-powered system that recommends sustainable packaging materials using machine learning, cost analysis, and CO₂ impact prediction.

## Completed Modules
- Module 1: Data Collection & Management
- Module 2: Data Cleaning & Feature Engineering
- Module 3: Machine Learning Dataset Preparation
- Module 4: AI Models (Random Forest & XGBoost) + Recommendation System

## Model Performance
- Cost Prediction R² ≈ 0.99
- CO₂ Prediction R² ≈ 0.99

## How to Run
pip install -r requirements.txt
python scripts/train_models.py
